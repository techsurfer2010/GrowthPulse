========================================================
DHT22 Temperature / Humidity Sensor
========================================================

              DHT22 Module

              VCC ─────────────── 3V3

              GND ─────────────── GND

             DATA ─────────────── GPIO5


========================================================
Soil Moisture Sensor Module
========================================================

          Soil Moisture Module

              VCC ─────────────── 3V3

              GND ─────────────── GND

                AO ─────────────── GPIO1

                DO ─────── NOT USED


========================================================
Pin Usage Summary
========================================================

GPIO4  → DS18B20 DATA
GPIO5  → DHT22 DATA
GPIO1  → Soil Sensor Analog Output

3V3    → All sensor VCC
GND    → All sensor grounds


========================================================
Important Notes
========================================================

1. DS18B20 REQUIRES a 4.7kΩ pull-up resistor
   between DATA and 3V3.

2. Soil sensor uses ANALOG output only (AO).

3. DO NOT power sensors from 5V.

4. ESP32-S3 GPIO is 3.3V ONLY.

5. All sensor grounds MUST be common.
